# Spotify Top 50 Dashboard – Power BI

A comprehensive Power BI dashboard analyzing Spotify's Global Top 50 chart data, designed for music analysts, playlist managers, and marketing teams to monitor song and artist performance.

## 📊 Dashboard Pages

**Overview Page**
- Key KPIs: Total Songs, Distinct Artists, Avg Popularity, Avg Duration
- Explicit vs Non-Explicit song comparison and share analysis
- Songs by Album Type (single, album, compilation)
- Yearly trends: Distinct Songs and Avg Popularity
- Monthly trend analysis for Popularity and Distinct Songs
- Top Songs & Top Artists by Popularity

**Artist Page**
- Top Artists by Popularity ranking
- Tracks per Album vs Songs by Artist comparison
- Drill-down to artist-level data: songs, release dates, avg popularity, avg position, duration
- Identify artists with consistent hits and #1 positions

**Songs Page**
- Top Songs by Popularity ranking
- Tracks per Song distribution (Album vs Single)
- Detailed table: song name, release date, distinct artists, avg popularity, position, duration by year

## ❓ Problem Solved

Raw Spotify "Top 50" data provides only lists and rankings, making pattern identification difficult. This dashboard enables:
- ✅ Quick KPI monitoring for key metrics
- ✅ Explicit vs non-explicit content performance analysis
- ✅ Song/album distribution tracking by type and year
- ✅ Trend visibility through monthly/yearly tracking
- ✅ Connected insights with drill-down from overview to details
- ✅ Data-driven decisions for promotion and curation

## 🛠️ Built With

- **Power BI Desktop** with DAX Query View for efficient measure management
- **30+ Custom DAX Measures** using CALCULATE, DISTINCTCOUNT, AVERAGE, time-intelligence functions
- **Interactive Visualizations**: KPI cards, bar charts, donut charts, matrix tables, drill-through pages

## 📁 Dataset

Spotify Global Top 50 chart data including:
- Song/artist names, popularity scores, duration (ms)
- Explicit content flags, album types, chart positions, release dates

## 🚀 Getting Started

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Navigate through Overview, Artist, and Songs pages
4. Click visuals to cross-filter and explore drill-through features
